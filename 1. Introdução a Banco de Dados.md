# 1. Introdução a Banco de Dados

## 1.1 Conceitos Básicos

### Banco de Dados (DB)

- Conjunto organizado de informações.
    
- Utilizado para armazenar dados de forma estruturada.
    
- Deve permitir consultas, alterações e recuperação de informações.
    

### SGBD (Sistema de Gerenciamento de Banco de Dados)

- Software responsável por criar, organizar e administrar bancos de dados.
    
- Fornece ferramentas para inserir, consultar, atualizar e remover dados.
    
- Atua como intermediário entre as aplicações e o banco de dados.
    

Exemplos:

- SQLite
    
- MySQL
    
- PostgreSQL
    
- SQL Server

## 1.2 Arquitetura Cliente-Servidor
### Server

- Máquina ou processo responsável por disponibilizar serviços.
    
- No contexto de banco de dados, hospeda e executa o SGBD.
    
- Recebe solicitações dos clientes e retorna os resultados.
    

### Client

- Aplicação que consome os dados do banco.
    
- Envia comandos e consultas ao servidor.
    
- Exibe ou manipula as informações recebidas.
    

Exemplos:

- Aplicações Web
    
- Aplicações Desktop
    
- APIs
    
- Aplicativos Mobile
