# 1. Introdução

> Local onde é armazenado os **metadados**.

- Metadados são dados *sobre* os dados;
- Todas as **informações** *acerca* dos dados;
- Essa informações ficam persistidas para orientar o SGBD;

# 2. Modelagem de Dados

- O dicionário de dados fica entre a *modelagem conceitual* e *modelagem lógica*;

# 3. Show Table e Show Create Table

- Usado para quando o dicionário de dados for feito após a modelagem física;
- `show table`: mostra as tabelas do banco;
- `show create table`: mostra o comando de criação daquela tabela;
