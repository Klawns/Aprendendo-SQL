# 1. Introdução
Agora vamos sair do conjunto de DDL, agora vamos para o DML e DQL

- DML é o conjunto para manipular dados
- DQL é o conjunto para consultar dados

# 2. INSERT  
  
```sql  
INSERT INTO tabela (coluna1, coluna2, coluna3)  
VALUES (valor1, valor2, valor3);  
```  
  
- Insere um novo registro na tabela.  
- A ordem das colunas pode ser qualquer uma.  
- Os valores devem seguir exatamente a mesma ordem das colunas.  
- Colunas com `DEFAULT` ou auto incremento podem ser omitidas.  
  
Exemplo:  
  
```sql  
INSERT INTO todos (title, completed)  
VALUES ('Estudar SQL', false);  
```  
  
---  
  
# 3. SELECT  
  
```sql  
SELECT colunas  
FROM tabela;  
```  
  
- Consulta informações armazenadas em uma tabela.  
  
Buscar todas as colunas:  
  
```sql  
SELECT *  
FROM todos;  
```  
  
Buscar apenas algumas colunas:  
  
```sql  
SELECT id, title  
FROM todos;  
```  
  
Buscar apenas uma coluna:  
  
```sql  
SELECT title  
FROM todos;  
```  
  
---  
  
# 4. WHERE  
  
```sql  
SELECT colunas  
FROM tabela  
WHERE condição;  
```  
  
- Filtra os registros retornados pelo `SELECT`.  
- Sem o `WHERE`, todas as linhas da tabela são retornadas.  
  
## Operadores de comparação  
  
| Operador | Significado    |
| -------- | -------------- |
| =        | Igual          |
| != ou <> | Diferente      |
| >        | Maior          |
| <        | Menor          |
| >=       | Maior ou igual |
| <=       | Menor ou igual |
  
Exemplos:  
  
```sql  
SELECT *  
FROM todos  
WHERE id = 3;  
```  
  
```sql  
SELECT *  
FROM todos  
WHERE completed = true;  
```  
  
```sql  
SELECT *  
FROM todos  
WHERE id > 5;  
```  
  
---  
  
## Operadores Lógicos  
  
### AND  
  
Todas as condições devem ser verdadeiras.  
  
```sql  
SELECT *  
FROM todos  
WHERE completed = true  
AND id > 5;  
```  
  
---  
  
### OR  
  
Pelo menos uma condição deve ser verdadeira.  
  
```sql  
SELECT *  
FROM todos  
WHERE id = 1  
OR id = 5;  
```  
  
---  
  
### NOT  
  
Inverte uma condição.  
  
```sql  
SELECT *  
FROM todos  
WHERE NOT completed = true;  
```  
  
Equivale a:  
  
```sql  
SELECT *  
FROM todos  
WHERE completed = false;  
```  
  
---  
  
## BETWEEN  
  
Verifica se um valor está dentro de um intervalo.  
  
```sql  
SELECT *  
FROM todos  
WHERE id BETWEEN 5 AND 10;  
```  
  
Equivale a:  
  
```sql  
WHERE id >= 5  
AND id <= 10;  
```  
  
Também funciona com datas.  
  
```sql  
SELECT *  
FROM pedidos  
WHERE data BETWEEN '2025-01-01' AND '2025-12-31';  
```  
  
---  
  
## IN  
  
Verifica se um valor pertence a uma lista.  
  
Ao invés de:  
  
```sql  
WHERE id = 1  
OR id = 3  
OR id = 8;  
```  
  
Podemos fazer:  
  
```sql  
SELECT *  
FROM todos  
WHERE id IN (1, 3, 8);  
```  
  
Também funciona com textos.  
  
```sql  
SELECT *  
FROM usuarios  
WHERE cidade IN ('São Paulo', 'Rio de Janeiro');  
```  
  
---  
  
## LIKE  
  
Procura textos seguindo um padrão.  
  
### %  
  
Representa qualquer quantidade de caracteres.  
  
Contém:  
  
```sql  
WHERE title LIKE '%SQL';  
```  
  
Começa com:  
  
```sql  
WHERE title LIKE 'SQL%';  
```  
  
Termina com:  
  
```sql  
WHERE title LIKE '%SQL';  
```  
  
### _  
  
Representa exatamente um caractere.  
  
```sql  
WHERE nome LIKE 'J_o';  
```  
  
Resumo:  
  
- `%` → qualquer quantidade de caracteres.  
- `_` → exatamente um caractere.

# 5. ORDER BY  
  
```sql  
SELECT *  
FROM todos  
ORDER BY title ASC;  
```  
  
- Ordena os resultados retornados pelo `SELECT`.  
  
Tipos de ordenação:  
  
- `ASC` → Crescente (padrão).  
- `DESC` → Decrescente.  
  
Exemplos:  
  
```sql  
SELECT *  
FROM todos  
ORDER BY id ASC;  
```  
  
```sql  
SELECT *  
FROM todos  
ORDER BY id DESC;  
```  
  
```sql  
SELECT *  
FROM todos  
ORDER BY title ASC;  
```  
  
---  
# 6. UPDATE  
  
```sql  
UPDATE tabela  
SET coluna = valor  
WHERE condição;  
```  
  
- Atualiza um ou mais registros da tabela.  
- O `WHERE` define quais registros serão alterados.  
- Se o `WHERE` for omitido, **todos os registros da tabela serão atualizados**.  
  
Atualizando apenas uma coluna:  
  
```sql  
UPDATE todos  
SET completed = true  
WHERE id = 3;  
```  
  
Atualizando várias colunas:  
  
```sql  
UPDATE todos  
SET  
title = 'Estudar SQL',  
completed = false  
WHERE id = 5;  
```  
  
---  
  
# 7. DELETE  
  
```sql  
DELETE FROM tabela  
WHERE condição;  
```  
  
- Remove registros da tabela.  
- O `WHERE` define quais registros serão removidos.  
- Se o `WHERE` for omitido, **todos os registros da tabela serão apagados**.  
  
Removendo um registro:  
  
```sql  
DELETE FROM todos  
WHERE id = 3;  
```  
  
Removendo vários registros:  
  
```sql  
DELETE FROM todos  
WHERE completed = true;  
```  
  
Removendo todos os registros da tabela:  
  
```sql  
DELETE FROM todos;  
```
