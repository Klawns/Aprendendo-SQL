# Aprendendo SQL
- Repositório dedicado às minhas anotações e exemplos de SQL, utilizados durante os estudos de banco de dados relacionais. As anotações em Markdown são organizadas para uso no Obsidian.

# Estrutura do projeto
- Exemplos SQL/: contém exemplos organizados por capítulos. A numeração das pastas indica o tema abordado.
- !Imagens/: armazena imagens, capturas de tela e outros recursos utilizados nas anotações.
